Meus Objetivos

O projeto "Meus Objetivos" é uma aplicação interativa que ajuda o usuário a acompanhar metas de desenvolvimento pessoal, como cursos, projetos e atualização de currículo. Através de uma interface simples, ele permite alternar entre diferentes objetivos, mostrando um contador de tempo regressivo para cada um, incentivando o progresso e a organização das tarefas.

A estrutura HTML organiza o site em uma seção principal com um título e botões que representam diferentes metas. Cada meta possui uma aba de conteúdo correspondente, com detalhes e um contador que exibe o tempo restante para concluir o objetivo. Essa estrutura facilita a navegação e destaca os diferentes focos de desenvolvimento do usuário.

O CSS aplica uma paleta de cores moderna, usando tons escuros e acentos em verde para destacar o conteúdo. Com o uso de flexbox, o layout se adapta a diferentes tamanhos de tela, e as bordas arredondadas e transições suaves nos botões criam uma experiência visual agradável e profissional.

O JavaScript controla a interatividade, permitindo que o usuário alterne entre as abas de objetivos e atualize o cronômetro em tempo real para cada meta. Com base em datas futuras definidas no código, o contador exibe os dias, horas, minutos e segundos restantes, criando um senso de urgência e motivação.

Em resumo, o site "Meus Objetivos" oferece uma ferramenta prática e envolvente para gerenciamento de metas, unindo HTML, CSS e JavaScript para criar uma experiência completa e funcional. Ele facilita o acompanhamento do progresso e torna o planejamento de atividades pessoais mais intuitivo e organizado.

Minha Área de Estudo

O site "Área de Estudo" é uma aplicação interativa que permite ao usuário descobrir sua afinidade com diferentes áreas do conhecimento: Humanas, Exatas e Biológicas. A experiência é baseada em uma série de perguntas que orientam o jogador a escolher respostas que refletem seu perfil acadêmico. No final, o site exibe o resultado indicando a área com a qual o usuário mais se identifica.

A estrutura HTML organiza o conteúdo em seções funcionais. Um cabeçalho com o título “Qual é a sua Área?” é seguido pela caixa principal, que abriga perguntas, alternativas de resposta e o resultado final. O HTML também define a inclusão de imagens que reforçam o tema educacional, como ícones e representações visuais das áreas de estudo.

O CSS proporciona um visual limpo e moderno ao site. Ele utiliza uma paleta de cores com tons sóbrios para o fundo e cores mais vibrantes para destacar botões e textos. As variáveis CSS ajudam a padronizar essas cores, criando uma interface consistente e agradável. O layout responsivo e os botões com efeito de transição tornam a experiência mais envolvente e amigável ao usuário.

O JavaScript traz vida ao site, gerenciando a apresentação das perguntas e a pontuação de acordo com as escolhas do usuário. As perguntas, organizadas em arrays de objetos, contêm textos descritivos e opções de resposta. Cada resposta acumula pontos específicos que determinam a área de afinidade do usuário. Ao finalizar, o JavaScript exibe uma imagem correspondente à área de estudo e o resultado textual com base na pontuação.

Em resumo, o site “Área de Estudo” é uma ferramenta educativa e interativa, que explora o perfil acadêmico do usuário de forma dinâmica. A combinação de HTML, CSS e JavaScript cria uma experiência fluida, permitindo que o usuário navegue facilmente entre perguntas e receba feedback imediato sobre suas preferências de estudo.

Flash Cards

O projeto "FlashCard" é uma aplicação interativa que permite ao usuário revisar conteúdos de maneira dinâmica. Ele apresenta cartões de estudo com perguntas e respostas sobre temas variados, proporcionando uma forma prática de aprendizado e retenção de informações. O usuário pode alternar entre a pergunta e a resposta ao clicar no cartão, tornando o processo de estudo mais intuitivo e interativo.

A estrutura HTML organiza o conteúdo em um container principal, onde os cartões de estudo são exibidos. Cada cartão é criado dinamicamente com base em uma lista de perguntas e respostas, oferecendo uma experiência visual simples e organizada. Essa estrutura facilita o carregamento e a atualização dos cartões, mantendo o foco no conteúdo apresentado.

O CSS estiliza o site com um tema moderno, usando tons de azul e verde para destacar o conteúdo. Os cartões possuem um efeito de rotação que, ao clicar, alterna entre a pergunta e a resposta. O layout é responsivo, ajustando-se bem a diferentes tamanhos de tela para proporcionar uma experiência de uso confortável tanto em dispositivos móveis quanto em desktop.

O JavaScript controla a criação e a interatividade dos cartões. A função criaCartao gera novos cartões com base nas perguntas e respostas definidas em um arquivo externo. O código também lida com o evento de clique que ativa a rotação do cartão, exibindo a resposta ou retornando à pergunta. Esse recurso torna o estudo dinâmico e mantém o usuário engajado no conteúdo.

Em resumo, o site "FlashCard" é uma ferramenta educativa prática, que utiliza HTML, CSS e JavaScript para oferecer uma experiência de estudo interativa e acessível. A estrutura modular e o design intuitivo permitem que o usuário aprenda e revise conteúdos de forma eficiente, explorando temas variados de maneira envolvente.
